## ✅ 1. **Highlight per pulsante attivo nella navbar**

Aggiungi questo CSS per far capire su che sezione si trova l’utente:

### 🔧 Aggiungi dentro lo `<style>`:

```css
nav a:focus, nav a:hover {
  background-color: var(--primary);
  color: white;
}
```

---

## ✅ 2. **Scroll morbido e autochiusura del menu su mobile**

Evita salti bruschi quando si clicca un link della navbar.

### 🔧 Aggiungi nello `<head>` subito dopo lo `<script>`:

```html
<script>
  document.querySelectorAll('a[href^="#"]').forEach(anchor => {
    anchor.addEventListener('click', function (e) {
      e.preventDefault();
      const section = document.querySelector(this.getAttribute('href'));
      if (section) section.scrollIntoView({ behavior: 'smooth' });
    });
  });
</script>
```

## ✅ 5. **Effetto zoom on hover sulle immagini camere**

Rende la galleria visivamente più dinamica.

### 🔧 Aggiungi nel tuo `<style>`:

```css
.swiper-slide img {
  transition: transform 0.4s ease;
}
.swiper-slide img:hover {
  transform: scale(1.03);
}
```
